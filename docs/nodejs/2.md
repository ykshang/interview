# 常用的命令

- 查看 NPM 下载的全局模块的安装路径

  ```bash
  npm get prefix
  # C:\Users\53563\AppData\Roaming\npm

  ```

- 查看 NPM 下载的全局模块的安装路径

  ```bash
  npm config set prefix "D:\software\installation\nodejs\node_global"
  ```

- 查看 NPM 缓存的存放路径

  ```bash
  npm get cache
  ```

- 修改 NPM 缓存的存放路径
  ```bash
  npm config set cache "D:\software\installation\nodejs\node_cache"
  ```
