# Vue2和Vue3的区别

# vue2响应式原理

当你把一个普通的 JavaScript 对象传入 Vue 实例作为 data 选项，Vue 将遍历此对象所有的 property，并使用 Object.defineProperty (opens new window) 把这些 property 全部转为 getter/setter (opens new window)。Object.defineProperty 是 ES5 中一个无法 shim 的特性，这也就是 Vue 不支持 IE8 以及更低版本浏览器的原因。

Object.defineProperty(parent, "chrilren", {
  get() {
    return bValue;
  },
  set(newValue) {
    bValue = newValue;
  },
  enumerable: true,
  configurable: true,
});

缺点

  1. 对象：删除和添加属性无法识别
  2. 数组：直接修改数组长度；按照下标操作数组里的对象，vue识别不到

可以使用 $set 和 $delete 去处理，或者ES6 数组里的一些新方法比如Array.splice
